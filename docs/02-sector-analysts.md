# 02. 섹터 애널리스트 프레임워크 — GICS 11 + 매크로·금리

> 상태: 설계 초안 (2026-07-06) · 이 문서의 밸류체인·지표는 그래프 정적 계층(Layer 0)과 에이전트 시스템 프롬프트의 원천이 된다.

## 0. 공통 템플릿

모든 섹터 애널리스트는 동일한 구조를 갖는다:

| 항목 | 내용 |
|---|---|
| **담당 범위** | GICS 섹터 및 산업그룹 |
| **밸류체인** | 상류→하류 세그먼트 DAG. 각 세그먼트는 그래프의 `ValueChainSegment` 노드가 된다 |
| **수요·공급 드라이버** | 뷰 형성의 인과 골격 |
| **핵심 모니터링 지표** | 정기 관찰 데이터 시리즈 + 소스 (→ [04-data-pipeline.md](04-data-pipeline.md)) |
| **크로스섹터 엣지** | 타 섹터와의 인과 연결 (그래프 `AFFECTS` 엣지의 시드) |
| **대표 국내 상품 접점** | 이 섹터 노출을 제공하는 국내 상장 상품 유형 |

세그먼트 ID 규약: `{섹터코드}.{체인}.{단계}` 예: `IT.semi.equipment`. 그래프 스키마 문서에서 이 ID를 그대로 사용한다.

에이전트 산출물 3종(뷰·그래프 갱신 제안·모니터링 노트)과 기록 규칙은 [01-architecture.md §3-⑤](01-architecture.md) 참조.
애널리스트 내부 구조(스킬 라이브러리·리서치 메모리·두 축 학습 루프)는 12명 전원 공통이며 [05-analyst-structure.md](05-analyst-structure.md)에 정의한다.

## 1. 에너지 (Energy) — `EN`

**담당**: 석유·가스 시추/정제/유통, 에너지 장비·서비스

**밸류체인**
- `EN.oil`: 원유 도입(수입) → 정제(정유사) → 석유제품(휘발유·경유·항공유·납사) → 유통/수출
- `EN.gas`: LNG 도입 → 저장·기화 → 도시가스/발전용 공급

한국 특성: 상류(시추)가 거의 없고 **정제 중심**. 정유사의 이익 = 정제마진(크랙 스프레드)이 지배.

**수요·공급 드라이버**: 국제유가(WTI·Brent·두바이유), 정제마진, 글로벌 정제설비 증설/폐쇄, 항공유 수요 회복, 환율(원화 약세 = 도입 원가 상승), OPEC+ 감산

**핵심 모니터링 지표**
- WTI·Brent 유가 (FRED: `DCOILWTICO`, `DCOILBRENTEU`), 두바이유
- 싱가포르 복합 정제마진 (크랙 스프레드)
- 천연가스 (FRED: `DHHNGSP`, JKM LNG 현물가)
- USD/KRW 환율 (ECOS)

**크로스섹터 엣지**: 납사 → 소재(석유화학 원료), LNG → 유틸리티(발전 연료비), LNG선 발주 → 산업재(조선), 유가 → 매크로(물가)

**대표 상품 접점**: 에너지 섹터 ETF, 정유 개별주 편입 배당 ETF, 원유 선물 ETF(파생형)

## 2. 소재 (Materials) — `MA`

**담당**: 화학, 2차전지 소재, 철강·금속, 정밀화학, 건자재

**밸류체인**
- `MA.chem`: 납사 → 기초유분(에틸렌·프로필렌, NCC) → 중간재 → 합성수지/합성고무/화섬
- `MA.battery`: 광물(리튬·니켈·코발트) → 전구체 → 양극재/음극재/분리막/전해질 → 배터리 셀(→ 경기소비재의 완성차로)
- `MA.steel`: 철광석·원료탄 → 고로/전기로 → 판재·강관 → 자동차·조선·건설向
- `MA.nonferrous`: 구리·알루미늄 제련 → 전선·부품向

**수요·공급 드라이버**: 중국 경기(최대 수요처이자 최대 공급 경쟁자), 유가·납사(화학 원가), 리튬·니켈 가격, 전기차 판매량과 배터리 수주, 중국 철강 감산 정책, 미국 IRA/관세 정책

**핵심 모니터링 지표**
- 에틸렌-납사 스프레드, 리튬(탄산리튬)·니켈 가격
- 철광석·원료탄 가격, 중국 철강 가동률
- 글로벌 전기차 판매량, 중국 제조업 PMI
- 구리 가격 (경기 선행 지표 겸용)

**크로스섹터 엣지**: 에너지(납사 공급) → `MA.chem`, `MA.battery` → 경기소비재(전기차), `MA.steel` → 산업재(조선·건설), 원자재가 → 매크로(물가)

**대표 상품 접점**: 2차전지 테마 ETF, 소재 섹터 ETF, 철강·화학 편입 가치주 ETF

## 3. 산업재 (Industrials) — `IN`

**담당**: 조선, 방산, 기계·건설기계, 건설·엔지니어링, 전력기기, 항공·해운·물류

**밸류체인**
- `IN.ship`: 후판·기자재 → 블록 조립 → 건조(상선: 컨테이너선·LNG선·탱커) → 인도 (수주→인도 2~3년 시차)
- `IN.defense`: 부품·체계 → 완성 무기체계(자주포·미사일·항공기) → 내수/수출
- `IN.power_equip`: 변압기·차단기·전선 → 송배전망/데이터센터/신재생 연계
- `IN.construction`: 수주(주택·플랜트·인프라) → 시공 → 준공 (부동산 PF와 연결)
- `IN.transport`: 해운(운임) · 항공(여객/화물) · 물류

**수요·공급 드라이버**: 신조선가·수주잔고(조선), 글로벌 국방예산·지정학(방산), 전력망 투자·AI 데이터센터 전력 수요(전력기기), 국내 주택경기·해외 플랜트(건설), 운임지수(해운)

**핵심 모니터링 지표**
- 신조선가지수, 조선사 수주잔고
- SCFI(상하이컨테이너운임지수), BDI(발틱운임지수)
- 방산 수출 계약 공시, 각국 국방예산
- 국내 건설수주액 (ECOS), 미분양 주택 수
- 변압기 수출액 (관세청/무역통계)

**크로스섹터 엣지**: 소재(후판) → `IN.ship`, `IN.power_equip` → IT(데이터센터 증설)·유틸리티(송배전 투자), `IN.construction` → 부동산(PF)·금융(부동산 익스포저)

**대표 상품 접점**: 방산 테마 ETF, 조선 테마 ETF, 전력기기·AI인프라 테마 ETF, 산업재 섹터 ETF

## 4. 경기소비재 (Consumer Discretionary) — `CD`

**담당**: 자동차·부품, 유통(백화점·이커머스), 의류·잡화, 호텔·레저·카지노, 교육

**밸류체인**
- `CD.auto`: 부품(전장·샤시·배터리) → 완성차 → 내수/수출 판매 → 할부금융·A/S
- `CD.retail`: 브랜드/제조 → 유통 채널(백화점·마트·이커머스·면세) → 소비자
- `CD.leisure`: 여행·호텔·카지노·엔터테인먼트 소비

**수요·공급 드라이버**: 가계 실질소득·소비심리, 금리(할부·리볼빙 부담), 환율(수출차 채산성), 미국·유럽 자동차 수요, 중국 관광객(면세·카지노), 전기차 전환 속도

**핵심 모니터링 지표**
- 소비자심리지수 CCSI (ECOS), 소매판매액
- 완성차 수출·내수 판매량 (월별)
- 가계신용·연체율 (ECOS)
- 미국 소매판매 (FRED: `RSAFS`), 미시간 소비자심리 (FRED: `UMCSENT`)
- 방한 외국인 관광객 수

**크로스섹터 엣지**: 소재(배터리·강판) → `CD.auto`, IT(차량용 반도체·전장) → `CD.auto`, 금융(금리) → 소비 전반, `CD.retail` ↔ 필수소비재(채널 공유)

**대표 상품 접점**: 자동차 테마 ETF, 소비재 섹터 ETF, 리오프닝·여행 테마 ETF

---

## 5. 필수소비재 (Consumer Staples) — `CS`

**담당**: 음식료, 화장품·생활용품, 담배, 필수품 유통(편의점·슈퍼)

**밸류체인**
- `CS.food`: 곡물·원당·유지(수입 원재료) → 식품 가공 → 내수/수출(K-푸드)
- `CS.beauty`: 원료·ODM(코스맥스형 생산) → 브랜드 → 내수/면세/수출(중국·미국·일본·동남아)
- `CS.retail`: 편의점·기업형 슈퍼 (출점·점당 매출)

**수요·공급 드라이버**: 곡물가·환율(원가), 가격 전가력(인상 사이클), 중국 소비·따이공(화장품), K-푸드/K-뷰티 수출 모멘텀, 내수 인구구조

**핵심 모니터링 지표**
- 국제 곡물가 (옥수수·소맥·대두 — FRED/세계은행 원자재)
- 화장품·라면 등 품목별 수출액 (관세청 월별)
- CPI 식료품 (ECOS), 원/달러 환율
- 면세점 매출액

**크로스섹터 엣지**: 매크로(환율·곡물가) → `CS.food` 원가, 경기소비재(유통 채널·면세) ↔ `CS.beauty`, 소재(포장재) → `CS.food`

**대표 상품 접점**: 필수소비재 섹터 ETF, K-뷰티/K-푸드 테마 ETF, 저변동·배당 ETF(방어주 성격)

## 6. 헬스케어 (Health Care) — `HC`

**담당**: 바이오시밀러, CDMO, 신약개발, 제약(전통), 의료기기·미용기기, 진단

**밸류체인**
- `HC.pipeline`: 후보물질 발굴 → 전임상 → 임상 1/2/3상 → 허가(FDA·EMA·식약처) → 상업화/기술수출(L/O)
- `HC.cdmo`: 수주(글로벌 제약사) → 생산 캐파 증설 → 상업 생산
- `HC.biosimilar`: 오리지널 특허 만료 → 시밀러 개발·허가 → 미국·유럽 시장 침투
- `HC.device`: 의료·미용기기 개발 → 인허가 → 수출(소모품 매출 비중이 이익의 질 결정)

**수요·공급 드라이버**: 임상 이벤트(성공/실패가 개별 종목 주가의 지배 변수), FDA 승인·특허 만료 일정, 미국 생물보안법 류 지정학(중국 CDMO 대체 수혜), 금리(미실현 이익 성장주의 밸류에이션 민감도), 기술수출 계약

**핵심 모니터링 지표**
- 주요 파이프라인 임상 일정·결과 (공시·ClinicalTrials.gov)
- FDA/EMA 승인 캘린더, 오리지널 의약품 특허 만료 일정
- CDMO 수주 공시, 캐파 증설 계획
- 미국 10년물 금리 (FRED: `DGS10`) — 밸류에이션 할인율

**크로스섹터 엣지**: 매크로(금리) → `HC.pipeline` 밸류에이션, IT(AI 신약개발·디지털헬스) → `HC.pipeline`, 정책(약가·규제) → 전반

**대표 상품 접점**: 바이오 테마 ETF, 헬스케어 섹터 ETF, 고위험 성향 질의의 성장 옵션

## 7. 금융 (Financials) — `FI`

**담당**: 은행·금융지주, 증권, 보험(생보·손보), 카드·캐피탈

**밸류체인** (금융은 물리적 체인 대신 **자금 흐름 체인**)
- `FI.bank`: 수신(예금) → 여신(가계·기업 대출) → NIM(순이자마진) + 비이자이익 → 주주환원
- `FI.securities`: 브로커리지(거래대금) + IB(발행·PF) + 운용 → 이익
- `FI.insurance`: 보험료 수입 → 자산운용(채권 중심) → 보험손익(IFRS17 CSM) + 투자손익, 자본규제 K-ICS
- `FI.credit`: 카드 결제액·할부/리스 → 조달금리와의 스프레드

**수요·공급 드라이버**: 기준금리와 금리 커브(NIM·보험 자본에 직결), 가계대출 규제·연체율, 증시 거래대금, 부동산 PF 익스포저, 밸류업(주주환원 확대) 정책

**핵심 모니터링 지표**
- 기준금리, 국고채 3/10년, 예대금리차 (ECOS)
- 가계신용 잔액·연체율 (ECOS)
- KRX 일평균 거래대금
- 부동산 PF 연체율 (금감원 발표)
- 은행 배당수익률·주주환원율

**크로스섹터 엣지**: 매크로(금리) → `FI` 전반 (가장 강한 엣지), `FI.securities`·`FI.bank` → 부동산(PF), `FI.credit` → 경기소비재(소비 여력)

**대표 상품 접점**: 은행·금융 배당 ETF (인컴 질의의 단골), 증권 테마 ETF, 금융 섹터 ETF

## 8. IT (Information Technology) — `IT`

**담당**: 반도체(메모리·파운드리·소부장), 디스플레이, IT하드웨어·부품, 소프트웨어·IT서비스

**밸류체인**
- `IT.semi`: 설계(팹리스) → 소재(웨이퍼·포토레지스트·특수가스) / 부품 / 장비(전공정·후공정) → 제조(메모리: DRAM·NAND·HBM, 파운드리) → 후공정(OSAT·테스트) → 수요처(스마트폰·서버·AI 데이터센터)
  - 하위 세그먼트: `IT.semi.design`, `IT.semi.material`, `IT.semi.equipment`, `IT.semi.fab`, `IT.semi.memory`, `IT.semi.hbm`, `IT.semi.osat`
- `IT.display`: 소재(OLED 발광재료) → 패널 → 세트(TV·스마트폰·IT기기)
- `IT.hardware`: MLCC·기판·카메라모듈 → 세트 조립
- `IT.software`: SI·클라우드 MSP·보안

한국 시장의 최대 섹터. **메모리 사이클 + AI capex**가 지배 변수. 소부장(소재·부품·장비) 생태계가 두터워 세그먼트 세분화의 가치가 가장 크다.

**수요·공급 드라이버**: DRAM·NAND 고정거래가격, HBM 수요(북미 하이퍼스케일러 AI capex), 재고 사이클, 스마트폰·서버 출하량, 미·중 반도체 규제, 환율

**핵심 모니터링 지표**
- DRAM·NAND 고정가/현물가
- 한국 반도체 수출액 (관세청 월별 — 시장이 가장 주목하는 국내 지표)
- 필라델피아 반도체지수(SOX), 대만 전자업체 월매출
- 북미 빅테크 capex 가이던스 (분기 실적)
- 글로벌 반도체 매출 (WSTS/SIA)

**크로스섹터 엣지**: `IT.semi` → 커뮤니케이션(플랫폼의 AI 인프라 수요), IT(데이터센터) → 산업재(전력기기)·유틸리티(전력수요), `IT.hardware` → 경기소비재(전장), 매크로(환율) → 수출 채산성

**대표 상품 접점**: 반도체 테마 ETF(국내 최다 테마), AI·소부장 테마 ETF, IT 섹터 ETF, TR형 대형주 ETF(삼성전자·하이닉스 비중)

---

## 9. 커뮤니케이션서비스 (Communication Services) — `CO`

**담당**: 통신 3사, 인터넷 플랫폼, 게임, 미디어·엔터테인먼트

**밸류체인**
- `CO.telecom`: 네트워크 설비 투자(capex) → 무선(ARPU×가입자)·유선·기업(IDC·클라우드) → 배당
- `CO.platform`: 트래픽 → 광고·커머스·핀테크·콘텐츠 수익화, AI 투자
- `CO.game`: 개발(IP) → 퍼블리싱 → 국내외 매출(모바일·PC·콘솔)
- `CO.media`: 콘텐츠 제작(K-팝·드라마) → 국내외 유통(공연·음반·OTT) → 팬덤 플랫폼

**수요·공급 드라이버**: 통신은 규제·요금 정책과 배당 정책(디펜시브·인컴 성격), 플랫폼은 광고 경기와 AI 수익화, 게임은 신작 사이클, 엔터는 아티스트 활동 사이클과 해외 팬덤 확장

**핵심 모니터링 지표**
- 통신 3사 ARPU·배당수익률
- 디지털 광고 시장 성장률
- 게임 신작 출시 일정·앱마켓 순위
- 음반 판매량·공연 규모, 콘텐츠 수출액

**크로스섹터 엣지**: `CO.platform`·`CO.telecom`(IDC) → IT(AI 인프라 수요 생성), `CO.telecom` → 금융과 함께 인컴 상품군 구성, `CO.media` ↔ 경기소비재(소비 심리)

**대표 상품 접점**: 통신 배당 ETF(인컴), 게임·K-콘텐츠 테마 ETF, 플랫폼(인터넷) 테마 ETF

---

## 10. 유틸리티 (Utilities) — `UT`

**담당**: 전력, 가스 유틸리티, 지역난방, 신재생 발전

**밸류체인**
- `UT.power`: 연료 도입(LNG·석탄·우라늄) → 발전(기저: 원전·석탄 / 첨두: LNG / 신재생) → 송배전 → 판매(요금)
- `UT.gas`: LNG 장기계약 도입 → 도매 → 도시가스 소매

한국 특성: 요금이 정책 변수라 **이익 = f(연료비, 요금 정책)**. 규제 산업이며 배당·방어 성격.

**수요·공급 드라이버**: 전기요금·가스요금 인상 여부(정책), 유가·LNG 가격(연료비, SMP에 반영), 원전 가동률·신규 원전 정책, 데이터센터發 전력수요 증가, 미수금(가스공사) 회수 구조

**핵심 모니터링 지표**
- SMP(계통한계가격), 전력 수요량
- LNG 도입가·유가 (에너지 애널리스트와 공유)
- 전기·가스 요금 조정 발표 (정책 이벤트)
- 원전 이용률, 신재생 발전 비중

**크로스섹터 엣지**: 에너지(LNG·유가) → `UT.power` 연료비, IT(데이터센터) → 전력수요, `UT` → 산업재(송배전망 투자 = 전력기기 수주), 매크로(금리) → 고부채 유틸리티 이자 부담

**대표 상품 접점**: 유틸리티·배당 ETF, 원전 테마 ETF, 저변동 방어 ETF

---

## 11. 부동산 (Real Estate) — `RE`

**담당**: 상장 리츠, 부동산 인프라 펀드, 부동산 관련 운용

**밸류체인**
- `RE.reit`: 자산 편입(오피스·물류센터·리테일·주유소·데이터센터) → 임대수익 → 배당 분배 (자산유형별 공실률·임대료 사이클이 상이)
- `RE.fund`: 인프라 펀드(도로·항만 등 — 맥쿼리인프라형) → 통행료 등 수익 → 분배

한국 특성: 상장 리츠 시장이 작고(수십 개), **금리 민감도가 지배 변수**. 배당수익률 - 국고채 금리 스프레드로 밸류에이션.

**수요·공급 드라이버**: 금리(배당 스프레드·리파이낸싱 비용), 자산유형별 수급(오피스 공실률, 이커머스發 물류센터 수요, 데이터센터 신규 수요), 유상증자(주주가치 희석 이벤트), 부동산 PF 부실

**핵심 모니터링 지표**
- 국고채 3/10년 금리 (ECOS) — 배당 스프레드 계산
- 리츠별 배당수익률·LTV·리파이낸싱 일정
- 서울 오피스 공실률, 물류센터 공급량
- 부동산 PF 관련 지표 (금융 애널리스트와 공유)

**크로스섹터 엣지**: 매크로(금리) → `RE` 전반 (가장 강한 엣지), 금융(PF·대출) ↔ `RE`, 산업재(건설) → 공급, IT(데이터센터 리츠) 신규 접점

**대표 상품 접점**: 상장 리츠 개별 종목, 리츠 ETF, 인컴 질의의 채권 대체재

---

## 12. 매크로·금리 (Macro & Rates) — `MC` ★ GICS 외 추가

**담당**: 금리 커브, 크레딧, 환율, 물가, 글로벌 매크로. **채권 상품 추천의 주 엔진이자, 전 섹터 공유 컨텍스트 생산자.**

**분석 체계** (밸류체인 대신 전파 경로)
- `MC.rates`: 한은 기준금리 → 국고채 커브(3M~30Y) → 회사채·여전채 스프레드 → 상품 듀레이션·캐리 판단
- `MC.credit`: 신용등급별(AAA~BBB) 스프레드 → 크레딧 사이클 판단 → 회사채 상품 추천 가능 영역
- `MC.fx`: 원/달러 → 수출 섹터 채산성, 환헤지형 vs 언헤지형 상품 선택
- `MC.global`: 연준 금리·미 국채·달러지수·미국 CPI → 국내 금리·외국인 수급 전파

**핵심 모니터링 지표**
- ECOS: 기준금리, 국고채 수익률 전 구간, 회사채(AA-·BBB-) 수익률, CPI, 환율, 가계신용
- FRED: `DFF`(연방기금금리), `DGS2`/`DGS10`(미 국채), `T10Y2Y`(커브), `CPIAUCSL`, `DTWEXBGS`(달러지수), `VIXCLS`
- 금리 결정 캘린더 (한은 금통위, FOMC)

**출력 특이사항**: 섹터 뷰 대신 **커브 뷰(스티프닝/플래트닝), 듀레이션 스탠스, 크레딧 스탠스, 환율 뷰**를 생산한다.
이 뷰는 ① 채권·채권형 ETF 랭킹에 직접 가중되고 ② 타 섹터 애널리스트에게 공유 컨텍스트로 배포된다(예: "금리 인하 사이클 진입" → 금융·부동산·헬스케어 애널리스트의 뷰 갱신 트리거).

## 13. 섹터 간 인과 지도 (그래프 시드)

아래 엣지들은 그래프 Layer 0의 `AFFECTS` 엣지로 하드코딩한다 (방향·부호 포함):

| 원인 | → 결과 | 부호/내용 |
|---|---|---|
| MC.rates 금리↑ | FI.bank NIM | + (단기), 대출 부실 − (장기) |
| MC.rates 금리↑ | RE, HC.pipeline, CO.platform | − (밸류에이션·조달비용) |
| MC.fx 원화 약세 | IT.semi, CD.auto, CS 수출 | + (채산성) / 수입원가 − |
| EN 유가↑ | MA.chem 원가, UT 연료비, MC 물가 | − / − / 인플레 압력 |
| IT AI 데이터센터 증설 | IN.power_equip, UT 전력수요, CO.telecom IDC | + |
| CD 전기차 판매↑ | MA.battery 수요 | + |
| IN.construction 부진 | FI PF 부실, MA.steel 수요 | − |
| MC.global 미국 경기 | IT·CD·CS 수출 수요 | + |

이 지도는 "리튬 리스크 줄이려면 뭘 피해야 해?" 같은 역방향 질의와, "금리 인하 수혜 상품" 같은 매크로 질의의 탐색 경로가 된다.

## 14. 콜드스타트 구축 순서

1. 이 문서의 세그먼트 목록 → 그래프 Layer 0 노드·엣지로 입력 (수작업 + 스크립트)
2. 상품 마스터·구성종목 수집 (→ [04-data-pipeline.md](04-data-pipeline.md)) → 기업을 세그먼트에 매핑 (초기엔 GICS/KRX 업종분류로 근사, 이후 애널리스트가 세분화)
3. 섹터 애널리스트는 **IT·금융·매크로 3개부터** 가동 (질의 빈도·상품 커버리지가 가장 높은 영역) → 나머지 순차 확장
